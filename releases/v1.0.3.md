## ğŸ“¦ Installation

### Quick Install (No Node.js Required)

**Linux & macOS:**

```bash
curl -fsSL https://www.viho.fun/install.sh | sh
```

**Windows (PowerShell):**

```powershell
irm https://www.viho.fun/install.ps1 | iex
```

### Manual Download

Download the pre-built binary for your platform below:

- **Linux x64**: `viho-linux-x64`
- **Linux ARM64**: `viho-linux-arm64`
- **macOS Intel**: `viho-macos-x64`
- **macOS Apple Silicon**: `viho-macos-arm64`
- **Windows x64**: `viho-win-x64.exe`

After downloading, make it executable and move to your PATH:

```bash
# Linux/macOS
chmod +x viho-*
sudo mv viho-* /usr/local/bin/viho

# Windows
# Add the .exe to a directory in your PATH
```

### NPM Installation (Alternative)

If you already have Node.js installed:

```bash
npm install -g viho
```
